# Portforio_web

## 仮想環境の作成
```
python3 -m venv 環境の名前
```
## 仮想環境の有効化
```
. 環境の名前/bin/activate
```
## プロジェクトのダウンロード
```
git clone https://github.com/kimuraamu/Portforio_web.git
```
## フォルダ移動
```
cd Portforio_web
```
## パッケージを一括インストール
```
 pip install -r requirements.txt
```
## 開発サーバーの起動
```
python manage.py runserver
```
